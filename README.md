# Project 9: L'Oréal Routine Builder

L'Oréal is expanding what is possible with AI. This project upgrades the chatbot into a product-aware routine builder.

Users can:

- Browse real L'Oréal products by category
- Select products and build a custom routine
- Generate skincare and haircare routine guidance
- Ask follow-up questions in a chat interface
- Switch between multiple interface languages, including RTL languages
